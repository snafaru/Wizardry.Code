Wizardry Proving Grounds v3.1 - Pascal Code Changes
===================================================

The baseline is the re-engineered Pascal source code (and assembler code) from the Wizardry_I program by Thomas William Ewers. This project would not exist without it.


Enhancement #:	WC048 - Quickplot To Stop Update Hit Points of Poisoned Characters
==================================================================================

Status:		Fixed. Recompiled. Verified.

Date of change:	27-Jan-2026.

Contributors:	Eric Labelle.

Enhancements:	WC045 added code to automatically update the remaining hit points of poisoned characters in the status window while walking in the maze. This WC048 now checks if the game option Q)UICK PLOT is ON, if ON then the game engine stops updating the characters statistics and the screen draws much faster. Press Q)UICK PLOT again to turn it back OFF.
