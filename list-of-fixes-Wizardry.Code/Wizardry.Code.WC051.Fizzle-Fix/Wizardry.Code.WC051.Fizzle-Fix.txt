Wizardry Proving Grounds v3.1 - Pascal Code Changes
===================================================

The baseline is the re-engineered Pascal source code (and assembler code) from the Wizardry_I program by Thomas William Ewers. This project would not exist without it.


Bug/Issue #:	WC051 - Fizzle Fix
==================================

Status:		Fixed. Recompiled. Verified.

Date of change:	12-Feb-2026.

Contributors:	Eric Labelle.

Fix:		Game breaker fix. When stepping on a square which has the FIZZLES attribute, all spells for monsters and your party fail, even if in camp. The problem was that if you stepped on any normal square thereafter the FIZZLES effect did not stop. This meant for example if you got in a tough fight and needed to heal (or MALOR/Teleport out, he, he, he) you couldn't do it, and furthermore you could not cast spells in the next fight either, which often unfairly got your entire party slaughtered. With this fix, now stepping out of a warded FIZZLES square gives you party a chance to cast spells again and survive.
